# 모르는 용어 
1. 비구조화 할당 : 객체 안에 있는 값을 쉽게 변수/상수에 할당시키는 방법     
const {a,b} = object; object 변수(상수) 안에 있는 key a,b의     value 를 할당해줌 (배열에도 사용 가능)    

ex) 
const object = {a:1,b:2}    
const {a,b} = object
